El programa se encarga de crear una conexión a la base de datos con el nombre y la contraseña indicada en el código.
Se crea un menú en el cual se puede elegir uno de estos cuatro métodos: Insertar un nuevo empleado, ver todos los empleados que hay, actualizar el salario del empleado con una id especifica y eliminar a empleados por id. cada uno de estos métodos en funcionalmente igual a un ejercicio
anterior ya hecho. Funcionando esto como un "test"de los conocimientos adquiridos.
